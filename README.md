# DelArcoChasCarlosPMDM

## Aplicación Cartelera de Peliculas

###### La aplicación dispone de:

 1. Un Login
 2. Un Registro De Usuario
 3. Una Lista de Peliculas las cuales puedes añadir,eliminar o editar.
